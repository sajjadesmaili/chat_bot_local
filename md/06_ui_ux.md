Bilingual UX

- Support Persian and English equally
- Language toggle must be always visible (sidebar + settings)
- Switching language must flip layout direction (RTL for fa, LTR for en)
- Keep spacing, icons, and density consistent in both languages
- Persian typography must remain readable (avoid cramped labels)
- Prefer short UI labels; avoid truncated critical actions in either language
- Empty states, errors, placeholders, and shortcuts help text must be translated
