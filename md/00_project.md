# Project Overview

Goal:

Build a Local AI Chatbot.

Features

- Local
- Fast
- Modern
- Minimal
- Easy to maintain

The chatbot should support:

- OpenAI
- Ollama

Provider can be switched from settings.

The application should automatically detect available providers.

The UI should display current provider.

Support:

- Multiple Projects
- Multiple Chats
- Chat History
- Search
- Upload Documents
- Project Isolation
- RAG
- Logs
- Authentication (optional)
- Streaming Responses
- Bilingual Frontend (Persian + English)
- RTL / LTR layout switching
